jsQari - A JavaScript Quran Recitation Player

jsQari is an Open Source JavaScript based Quran Recitation player that was developed to be used for the Quran recitation feature of QuranTracker.com. It is a stand-alone Quran Recitation player that can be used on any website/application. It uses the Open Source jPlayer Media Player for playing the Quran ayahs in MP3 which are provided by Verse by Verse Quran. jsQari can be used stand-alone or integrated with a website/application using the simple API. Currently jsQari is thin on documentation. Just download the code and have a look at the source for index.html.
